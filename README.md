تو گھر والوں کی فکر چھوڑ دے،
آج صرف میرا دل سن لے،
میری چاہت کی گہرائی میں ڈوب جا،
جہاں صرف تم اور میں ہیں، اور کوئی نہیں۔
دن رات تیرے خیال میں جیتا ہوں،
تیرا دکھ، تیرا سکون سب میرا ہے،
یہ دنیا کچھ بھی کہے،
میری محبت کی شدت کم نہیں ہو سکتی۔
تو بس میرا ہاتھ پکڑ لے،
دوریاں، خوف، سب مٹ جائیں گے،
میں ہر لمحہ تیرا محافظ بنوں گا،
تیری ہر خوشی، تیرا ہر خواب، میرا وعدہ ہے۔
تو گھر والوں کی فکر چھوڑ دے،
کیونکہ یہ دل، یہ جان، یہ روح
صرف تیرے لیے دھڑکتی ہے،
صرف تیرے لیے جیتی ہے۔ ❤️
